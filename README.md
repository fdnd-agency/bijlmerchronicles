# Bijlmer Chronicles / Imaginaries Mapping

## About

Wij zijn een consortium van partners geworteld in Amsterdam-Zuidoost die zich inzetten om Amsterdam-Zuidoost beter op de hoogte te brengen. 
We werken samen om het bestaande werk van gidsen, kenners, kunstenaars en ondernemers te versterken die de waarde van deze buurt al hebben laten zien.

## Over het project

In dit project hebben we een wiki ontwikkeld om Zuid Oost te ontdekken.
Het platform biedt informatie over bezienswaardigheden in Zuid Oost en maakt het eenvoudig en toegankelijk voor bezoekers om nieuwe plekken te vinden.

## Ontwerpkeuzes

We hebben aan de homepage gewerkt, hieronder zie je de kleuren terug van Zuid Oost. We hebben een navbar gemaakt en een section met daarin Lemma's. Ook hebben wij de map toegevoegd in de homepage, waarin je bij het klikken de lemma's kan vinden.

# Installatie

- Clone de repo
```https://github.com/fdnd-agency/bijlmerchronicles.git```

- Installeer npm packeges 
```npm install```

- Run local dev server
```sh
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

> To deploy your app, you may need to install an [adapter](https://svelte.dev/docs/kit/adapters) for your target environment.
